# 🧠 Reconocimiento de Emociones Faciales con Deep Learning

Proyecto final de Inteligencia Artificial para la clasificación de 7 estados emocionales utilizando Redes Neuronales Convolucionales (CNN).

## 📋 Requisitos del Proyecto
* **Dataset**: 28.709 imágenes en escala de grises (48x48 píxeles).
* **Tecnologías**: Python, PyTorch, Streamlit, Pandas, OpenCV.
* **Precisión alcanzada**: 51.35% en el conjunto de test.

## 🚀 Instrucciones de uso local
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar la aplicación: `python -m streamlit run app.py`

## 🛠️ Estructura del repositorio
* `analisis_exploratorio.ipynb`: Cuaderno con todo el proceso de limpieza y entrenamiento.
* `app.py`: Código de la aplicación web interactiva.
* `modelo_emociones.pth`: Pesos del modelo entrenado.
* `requirements.txt`: Librerías necesarias para ejecutar el proyecto.
